ID: BR-BOOKING-001
Заголовок:
Возможна отправка формы бронирования питомца без выбора города

Предусловие:
Пользователь находится на странице «О нас» с формой бронирования питомца.

Шаги воспроизведения:
1.Перейти на страницу с формой бронирования питомца
2.Заполнить все обязательные поля валидными данными
3.Не выбирать значение в поле «В каком городе будет жить собачка»
4.Нажать кнопку отправки формы

Ожидаемый результат
Форма не должна отправляться.
Появляется сообщение об ошибке валидации поля «Город» (например: «Поле обязательно для заполнения»).

Фактический результат
Форма успешно отправляется без выбора города.
Сообщение об ошибке отсутствует.

Серьезность:

Средняя

Приоритет:

Средний
