# QA Pet Project — Guru QA Hacking

## Project Description
Учебный QA pet-проект по ручному тестированию веб-сайта  
https://guru.qahacking.ru/

Цель проекта — отработка навыков manual QA и формирование портфолио.

## Scope of Testing
В рамках проекта было протестировано:
- Доступность и загрузка сайта
- Основные пользовательские сценарии
- Формы и валидация данных
- Негативные сценарии
- Smoke-тестирование

## Test Artifacts
В репозитории представлены следующие артефакты тестирования:
- Checklists (чек-листы)
- Test Cases (тест-кейсы)
- Bug Reports (баг-репорты)

## Tools & Technologies
- Manual testing
- Chrome DevTools
- GitHub

## Result
Составлены чек-листы и тест-кейсы, выявлены и задокументированы дефекты.
