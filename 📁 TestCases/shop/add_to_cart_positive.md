# Test Case: Добавление товара в корзину

Предусловие:
Пользователь находится на странице магазина https://guru.qahacking.ru/index.php/magazin

Шаги:
1. Открыть страницу магазина
2. Выбрать любой товар
3. Нажать кнопку "Добавить в корзину"
4. Перейти в корзину

Ожидаемый резултат:
Выбранный товар отображается в корзине
